SkyFactory One - Windows Client Setup

Set up a Windows client to play SkyFactory One multiplayer with a Mac host running the Metal terrain renderer.

Requirements

ComponentRequirement
OSWindows 10 or 11
MinecraftJava Edition (purchased)
RAM8 GB minimum (4 GB allocated to Minecraft)
JavaInstalled automatically by CurseForge
NetworkSame LAN as the host machine

Setup Steps

1 Install CurseForge

Download and install the CurseForge app:

Download CurseForge

Run the installer and sign in with your Microsoft account when prompted.

2 Select Minecraft

Open CurseForge and select Minecraft as your game. If this is your first time, CurseForge will detect or ask you to install the Minecraft launcher.

3 Install SkyFactory One

  1. Click Browse Modpacks in the left sidebar
  2. Search for "SkyFactory One"
  3. Click on SkyFactory One by Darkosto
  4. Click Install
  5. Wait for the download to complete (~200 MB, 109 mods)
SkyFactory One uses Minecraft 1.16.5 with Forge 36.2.34. CurseForge handles all of this automatically.

4 Allocate More RAM

SkyFactory One needs more RAM than the default 2 GB:

  1. In CurseForge, right-click the SkyFactory One instance
  2. Click Profile Options
  3. Check the box for Memory Settings to override
  4. Set Allocated Memory to 4096 MB (or 6144 MB if you have 16+ GB RAM)
  5. Click Done

5 Launch the Game

  1. Click Play on the SkyFactory One instance
  2. Wait for Forge to load all 109 mods (first launch takes 2-5 minutes)
  3. You should see the Minecraft main menu with the SkyFactory One logo

6 Connect to LAN Server

Once the Mac host has opened a world to LAN:

  1. From the main menu, click Multiplayer
  2. The LAN game should appear automatically in the server list
  3. If it doesn't appear, click Direct Connect and enter the host's IP and port (shown in the host's chat when they open to LAN)
The host opens to LAN from in-game: Esc → Open to LAN → choose settings → Start LAN World. The chat will show something like "Local game hosted on port 25565".

Hosting from the Mac

On the Mac running the Metal terrain renderer:

  1. Launch SkyFactory One (via CurseForge or the optimized launch-skyfactory.sh)
  2. Create or load a SkyFactory world
  3. Press Esc → Open to LAN
  4. Choose game mode and whether to allow cheats
  5. Click Start LAN World
  6. Note the port number shown in chat
  7. Tell the Windows player to connect to 192.168.4.222:<port>

Troubleshooting

Can't see the LAN game

  • Both machines must be on the same network/subnet
  • Check Windows Firewall: allow javaw.exe through (both inbound and outbound)
  • On Mac, check System Settings → Network → Firewall is not blocking Java
  • Try direct connect with the exact IP:port instead of relying on auto-discovery

Connection refused

  • Verify the host has opened to LAN (check chat for the port message)
  • Ping the host from Windows: ping 192.168.4.222
  • Ensure both clients are running the same SkyFactory One version

Mod mismatch / kicked on join

  • Both machines must have the exact same modpack version installed
  • Do not add or remove mods on either side
  • The Metal terrain renderer mod is client-side only and does not affect compatibility

Out of memory / crashes

  • Increase allocated RAM to 6144 MB in CurseForge profile options
  • Close other applications (browsers, etc.) to free RAM
  • If using Java arguments manually, set -Xmx6G -Xms4G

Low FPS on Windows

  • The Metal terrain renderer only works on macOS/Apple Silicon
  • On Windows, install OptiFine for 1.16.5 for better FPS
  • Reduce render distance to 8-10 chunks
  • Turn off smooth lighting and fancy graphics

In-Game Controls (Mac Host Only)

F6Toggle profiler overlay (per-stage timing, GPU stats)
F8Toggle Metal terrain on/off (A/B compare vs GL)

These controls only apply to the Mac client running the Metal terrain renderer mod.
